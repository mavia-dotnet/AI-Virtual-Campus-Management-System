# AI Virtual Campus Management System

## Overview
AI Virtual Campus Management System is an ASP.NET Core MVC based web application designed to manage campus activities digitally.

## Features
- Department Management
- Notice Management
- Event Management
- Feedback Management
- Authentication & Authorization
- Role Based Access Control
- SQL Server Database Integration

## Technologies Used
- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- HTML5
- CSS3
- Bootstrap
- JavaScript
